# Maksim-bot — Пошаговая инструкция запуска

> Для Артёма. Шаги выполняются один раз, в указанном порядке. Время — ~20 минут.

---

## Шаг 1. Создать TG-бота через @BotFather (3 мин)

В Telegram → [@BotFather](https://t.me/BotFather):

```
/newbot
```

В ответ на «How are we going to call it?»:
```
Юмсунов Контент-бот
```

В ответ на «Now let's choose a username»:
```
yumsunov_content_bot
```
*(если занято — попробуй `maxim_lifedrive_bot` или `yumsunov_realbiz_bot`)*

BotFather пришлёт **токен** вида `123456789:ABCdef...`. Скопируй его.

### Бонус — настройка через @BotFather (можно потом)

Описание для новых пользователей:
```
/setdescription
```
выбираешь созданного бота, потом текст:
```
Контент-бот для канала «Юмсунов | Про реальный бизнес». Бета — все артефакты на ревью перед публикацией.
```

Команды для меню (рядом с полем ввода):
```
/setcommands
```
выбираешь бота, вставляешь:
```
start - Главное меню
help - Справка
menu - Снова показать меню
cancel - Отменить текущее действие
brand - Сменить активный бренд
```

## Шаг 4. Получить NOTION_TOKEN (5 мин)

### Вариант A — новая интеграция (рекомендую — изолирует от Pit)

1. Открой [notion.so/my-integrations](https://www.notion.so/my-integrations)
2. **Важно:** проверь что в правом верхнем углу выбран workspace `Юмсунов Максим's Space`. Если выбран твой `Obyvka86` — переключи.
3. Нажми **`+ New integration`**
4. Заполни:
   - **Name:** `Maksim Content Bot`
   - **Logo:** не нужен
   - **Associated workspace:** `Юмсунов Максим's Space`
   - **Type:** `Internal`
5. Нажми **`Save`**
6. На следующем экране выбери **Capabilities**:
   - ✅ Read content
   - ✅ Update content
   - ✅ Insert content
   - ❌ User capabilities (не нужно)
   - ❌ Comment capabilities (не нужно)
7. **`Save changes`**
8. На вкладке **`Configuration`** скопируй **Internal Integration Token** (`ntn_…`)

### Подключить интеграцию к Content-странице

1. Открой страницу [Контент в Notion Максима](https://www.notion.so/3586889cd6a7804e9f0be2d58c34e872)
2. В правом верхнем углу нажми **`•••`** → **`Connections`**
3. Нажми **`+ Add connections`**
4. В поиске введи `Maksim Content Bot` → выбери → подтверди

### Вариант B — переиспользовать `Nox assist Max`

Если не хочешь возиться с новой интеграцией (но риск что один токен на 2 продукта):
1. Открой [my-integrations](https://www.notion.so/my-integrations)
2. Найди `Nox assist Max`
3. На вкладке `Configuration` → жми `Show` рядом с Internal Integration Token → копируй

## Шаг 5. Заполнить `.env` (2 мин)

В терминале на Windows:
```powershell
cd D:\AI\maksim-bot
copy .env.maksim.template .env
notepad .env
```

В Notepad заполни **минимум 4 переменные**:

```env
TELEGRAM_BOT_TOKEN=123456789:ABCdef...   # из шага 1
ADMIN_TELEGRAM_IDS=384671843              # из шага 3
NOTION_TOKEN=ntn_...                       # из шага 4
ANTHROPIC_API_KEY=sk-ant-...               # из твоего content-bot/.env
```

Опционально — добавь твоего chat_id канала и доп. AI-ключи (HeyGen/ElevenLabs/FAL) — но без них бот тоже стартует, просто без видео-генерации.

Сохрани и закрой Notepad.

## Шаг 6. Pre-flight check (1 мин)

```powershell
python setup_bot.py
```

Скрипт проверит **7 пунктов**:

1. `.env` найден
2. Все обязательные переменные заполнены
3. TG-токен валидный (запрашивает `getMe`)
4. Notion-токен валидный (запрашивает `/users/me`)
5. Notion DB подключена к интеграции
6. Anthropic API key валидный
7. Промпт-файлы Максима на месте

Если всё ✅ — иди дальше. Если ❌ — скрипт скажет что именно поправить.

## Шаг 7. Запустить бот (3 мин)

```powershell

```

Должно появиться (примерно):

Бот **слушает Telegram** в твоём терминале. Пока окно PowerShell открыто — бот работает.

## Шаг 8. Smoke-test первого ролика (5 мин)

В Telegram:

1. Открой своего нового бота (по `@yumsunov_content_bot` или как ты его назвал)
2. Жми **`Start`** или пиши `/start`
3. Должно появиться приветствие Максима с 4 кнопками:
   - 🎬 Виральный ролик
   - 📝 TG-пост (10 пунктов)
   - 🗣 Развёрнутая мысль
   - 😄 Юмор-помощник
4. Жми **`🎬 Виральный ролик`** → бот отвечает «Формат: Виральный. Пиши идею»
5. Пишешь:
   ```
   Как мы выбирали место под глэмпинг — реальные ошибки первого года
   ```
6. Бот отвечает «🔄 Пишу сценарий... ≈30 сек»
7. Через ~50 сек получаешь:
   - Сообщение со сценарием (JSON c hook overlay + 10 пунктами caption)
   - Сообщение с TG-постом
   - Сообщение «✅ Готово!» + ссылка на Notion-карточку

8. Открываешь ссылку Notion — видишь созданную карточку с full body.

**Если всё прошло — это первый рабочий цикл бота.** 🎉

## Что делать если что-то пошло не так

**Бот не отвечает в Telegram:**
- Проверь что окно `python bot.py` открыто и не упало
- Проверь что твой `chat_id` есть в `ADMIN_TELEGRAM_IDS`

**`Forbidden: bot was blocked by the user`:**
- Удалил себя из бота → отблокируй и нажми Start заново

**Notion-карточка не создалась:**
- Проверь что `Maksim Content Bot` (или используемая интеграция) подключена к Content-странице через Connections
- В терминале где бот — будет лог с traceback'ом

**Сценарий приходит «как от Артёма» (про AI и эксперимент):**
- В `.env` `DEFAULT_BRAND` должен быть `=maksim`. Проверь
- Перезапусти бот после правки `.env`
