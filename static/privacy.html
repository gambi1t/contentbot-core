Политика конфиденциальности

Последнее обновление: 11 апреля 2026

Настоящая Политика описывает, как приложение Panferov Content (далее — «Приложение»), оператором которого является Артём Панфёров (далее — «мы»), обрабатывает данные пользователей при интеграции с Instagram.

1. Какие данные мы получаем

Приложение подключается к Instagram Graph API и получает от Meta Platforms webhook-уведомления о следующих событиях, связанных с контентом Instagram-аккаунта владельца Приложения:

  • публичные комментарии к постам (идентификатор комментария, текст, имя пользователя автора, идентификатор поста);
  • входящие личные сообщения (Direct Messages), присланные владельцу аккаунта;
  • служебные события (подтверждение прочтения, переходы по ссылкам).

Приложение не собирает контактные данные, адреса электронной почты, телефоны или любую иную персональную информацию, которую пользователь не отправил явно в комментарии или сообщении.

2. Как мы используем данные

Полученные данные используются исключительно для автоматического ответа пользователю, оставившему комментарий с определённым ключевым словом, а именно: отправки ссылки на бесплатный информационный материал (гайд) в личные сообщения Instagram. Никаких других целей обработки нет.

3. Хранение данных

Приложение ведёт технический журнал обработанных событий (идентификаторы комментариев, имена пользователей, факт успешной отправки ответа), необходимый для предотвращения повторных ответов и диагностики ошибок. Журнал хранится на сервере владельца Приложения (Hetzner Online GmbH, Германия) и не передаётся третьим лицам.

Срок хранения журнала — не более 90 дней, после чего данные автоматически удаляются.

4. Передача третьим лицам

Мы не продаём, не передаём и не публикуем данные пользователей третьим сторонам. Данные обрабатываются только владельцем Приложения и остаются в пределах его серверной инфраструктуры.

5. Права пользователей

Любой пользователь имеет право запросить удаление связанных с ним данных из журналов Приложения. Порядок запроса описан на странице Инструкция по удалению данных.

6. Безопасность

Соединение между Instagram и Приложением защищено TLS. Webhook-запросы проверяются по подписи X-Hub-Signature-256 с использованием секрета приложения Meta — запросы без корректной подписи отклоняются.

7. Изменения в Политике

Мы можем обновлять Политику. Актуальная версия всегда доступна по этому адресу. Дата последнего обновления указана вверху страницы.
