# Idea seeds — банк зацепок для генератора идей Максима
#
# Формат: одна зацепка на строку, секции разделены `## SECTION`.
# Пустые строки и строки начинающиеся с `#` игнорируются генератором.
#
# Как используется:
#   idea_generator.py при каждом вызове рандомно выбирает 7 строк
#   (из всех секций равномерно) и инжектит их в user prompt как
#   подсказку: «Возьми один из этих углов как стартовую точку или
#   придумай свой». LLM либо использует, либо отвергает.
#
# Цель: разорвать детерминизм базового контекста Максима и дать
# свежий ветер в каждой партии — иначе Opus крутит одни и те же
# 5-6 концепций (делегирование, стратегия vs операционка и т.д.).
#
# Артём может править этот файл руками без перезапуска бота —
# кэш сбрасывается через `idea_generator.reload_idea_prompts()`
# или при рестарте сервиса.
